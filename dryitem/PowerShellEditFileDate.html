PowerShell 修改文件日期方法

方法 1：使用 Set-ItemProperty（最简单）

修改最后修改时间（LastWriteTime）：

Set-ItemProperty -Path "C:\test\file.txt" -Name LastWriteTime -Value "2024-01-01 10:00:00"

修改创建时间（CreationTime）：

Set-ItemProperty -Path "C:\test\file.txt" -Name CreationTime -Value "2024-01-01 10:00:00"

修改访问时间（LastAccessTime）：

Set-ItemProperty -Path "C:\test\file.txt" -Name LastAccessTime -Value "2024-01-01 10:00:00"

方法 2：使用 .NET 方法（最全面）

$file = Get-Item "C:\test\file.txt"
$file.CreationTime = "2024-01-01 09:00:00"
$file.LastWriteTime = "2024-01-02 12:00:00"
$file.LastAccessTime = "2024-01-03 15:00:00"

方法 3：一行命令同时改三个时间

(Get-Item "C:\test\file.txt").CreationTime = "2024-01-01 08:00:00"
(Get-Item "C:\test\file.txt").LastWriteTime = "2024-01-01 08:00:00"
(Get-Item "C:\test\file.txt").LastAccessTime = "2024-01-01 08:00:00"

日期格式注意事项

  • "2024-01-01 12:00"
  • "2024/01/01 12:00"
  • [datetime]::Parse("2024-01-01T12:00:00")

批量修改文件夹内所有文件日期

Get-ChildItem "C:\test" | ForEach-Object {
    $_.LastWriteTime = "2024-01-01 00:00:00"
}
